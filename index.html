手动安装也简单

7 步完成，每一步都有详细说明

1

安装 Termux

从 F-Droid 下载 Termux（不要用 Google Play 版），安装后运行 pkg install nodejs python openssh

2

服务器预下载

在服务器运行 npm install openclaw --ignore-scripts，跳过慢网络和 git 依赖问题

3

打包传输

精简后打包 97MB，通过 SSH 传输到手机，10 秒搞定

4

应用补丁

运行 scripts/patch-android.sh，修复 Android 不支持 systemd 的问题

5

配置 OpenClaw

写入 ~/.openclaw/openclaw.json，配置 AI 模型和飞书通道

6

启动 Gateway

在 Termux App 中运行 openclaw gateway run（⚠️ 必须从 App 启动，不能从 SSH）

7

飞书配对

首次发消息会收到配对码，运行 openclaw pairing approve feishu XXXX 完成配对
